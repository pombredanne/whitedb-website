whitedb-0.7.1.tar.gz

MD5 fingerprint: 5286505322089c0766255aaed8ebff28

The 0.7.1 release contains the following improvements since 0.7.0:

  • 64-bit Windows builds fixed (thanks malkia)
  • fixed index metadata initialization
  • updated the JSON parser (new version comes with a FOSS compatible license)
  • documentation updates

The full change log is available via Github.

The Github main branch is used for active development. You will find unfinished components and features.

There is no configure script on the main branch: use the provided Bootstrap script running aclocal, autoheader, autoconf, libtoolize and automake to build the configure script.
